* 클래스와 객체

클래스의 정의 :객체를 정의한것

용도 : 객체를 생성할때 사용

객체 정의 : 존재하는 사물이나 개념

용도 : 객체가 보유한 기능과 속성

클래스 ex) 설계도
객체 ex) 제품 : TV , 에어컨 등등

객체 = 속성(변수) + 기능(메소드)

속성 : 크기 , 길이 , 높, 색 등

기능 : 켜기 ,끄기 , 불륨 높이기

* 객체와 인스턴스

객체 : 인스턴스를 대표하는 용어

인스턴스 : 특정클래스로부터 생성된 객체

설계도 -> 인스턴스 (객체)

* 객체 생성법

클래스명 / 변수명

변수명 = new 클래스명 ();

**클래스 객체 생성, 객체 주소 참조 변수에 저장**

Tv t;
**참조변수 (리모컨 <객체>)**

t = new Tv();
**인스턴스 생성 후 생성된 Tv 인스턴스의 주소 t에 저장**

t.channel = x;
**Tv 인스턴스 멤버변수 channel 의 값을 x로함**

Tv t;
t = new Tv();

t.channel = 7;
t.channelDown();
