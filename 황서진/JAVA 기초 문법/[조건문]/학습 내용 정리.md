# 1. 조건문이란?

	조건의 결과에 따라서 양자 택일 또는 다자 택일을 진행한다.

* 주로 양자택일에는 if문, 다자 택일에는 switch문이 쓰인다. (반드시 그런 건 아님!) 
   

# 2. if문

	기본 자료형 데이터를 담고 있는 변수와 달리 배열 변수는 배열 데이터의 주소를 담고 있다.
	
>**if(조건식)**    
>**if(조건식) else**    
>**if(조건식) else if(조건식) else**
