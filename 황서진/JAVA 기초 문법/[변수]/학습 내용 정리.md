#2. 변수 선언과 초기화

	int i; // 자료형과 변수 이름을 나열하여 변수 선언한다.
	i = 10; // 이렇게 초기값을 설정해 주는 것을 변수 초기화라 한다.
	


#3. 메모리 할당과 진법

	2진수 사용!  
	칸 하나당 1비트라고 생각하면 된다.
	4비트가 1바이트.
	* int 자료형은 메모리에서 4바이트 공간 차지
