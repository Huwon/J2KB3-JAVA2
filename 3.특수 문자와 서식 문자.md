## 특수문자


>**\t** : 탭

>**\n** : 줄바꿈

>**\'** : 작은 따옴표

>**\"** : 큰 따옴표

>**\\** : 역 슬래쉬

>/*
    */ : 주석

   + 주석은 컴파일러가 컴파일하지 않습니다.


## 서식문자

    일반 문자가 아닌 서식에 사용되는 문자
    printf() 메서드 이용 : f는 format(형식)을 뜻함

>**%d** : 10진수

>**%o** : 16진수

>**%c** : 문자

>**%s** : 문자열

>**%f** : 실수

+ 서식 문자를 이용해서 출력 문자의 정렬 및 소수점 제한 기능을 사용할 수 있다.
